# Portfolio Notes

## Recruiter Framing

This project demonstrates basic data structures, recursion, tokenisation, and console application design in C#.

## Best Talking Points

- Implemented a binary search tree from scratch.
- Used in-order traversal to produce alphabetically sorted output.
- Stored frequency counts and source line numbers for each word.
- Identified the limitation of unbalanced trees and a clear improvement path.
